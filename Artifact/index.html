World Wide Web

The World Wide Web (W3) was invented by Tim Berners-Lee in 1989 and it is a system of interconnected pages of content accessed via the Internet using a web browser. A page of content, called a web page, can contain text, images, video and animation. Details about the W3 model can be found in Tim Berners-Lee's paper.

Why World Wide Web Succeed

Apart from the World Wide Web, there were many existing information systems, such as Alex, Gopher, Prospero and Xanada. But only the World Wide Web succeed and became the most popular method to access information that stored on different computers.

Compared with other existing information systems, the World Wide Web has many features.

  • The World Wide Web is well standarised, which means that it has common document naming schemes, common document formats and common network formats.
  • The data model involves in hypertext links and text searches.
  • The client-server model allows the user to choose differnet browser programs.
  • The World Wide Web is accessible from computers with different operating systems or platforms.
  • HTML is flexiable and not difficult to use. So, it is relatively easy to create HTML web pages.
  • The World Wide Web offers an easy-to-use interface, which means that users can have access to documents by clicking on hypertext links.
  • The World Wide Web offers the consistent user interface for all types of data.
  • The World Wide Web browser programs can offer access to Gopher, WAIS and other existing information systems, simply by storing their databases in a server program.

Due to these features, the World Wide Web can allow users to have access to documents in the servers from any browsing platform. Besides, almost all existing information systems can be represented in terms of the W3 model.
